# AI-Powered Market Sentiment Analyzer

An AI-powered financial sentiment analysis dashboard that analyzes market news headlines using VADER NLP and generates bullish/bearish insights.

## Features
- Market sentiment analysis
- Bullish/Bearish classification
- Downloadable reports
- Interactive dashboard UI
- Financial NLP analysis
- Market mood detection

## Tech Stack
- HTML
- CSS
- JavaScript
- VADER Sentiment Analysis
